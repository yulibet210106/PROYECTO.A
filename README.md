## Descripción

La "Lista de Tareas" es una aplicación web sencilla y efectiva que permite a los usuarios gestionar sus tareas diarias. Ofrece funcionalidades como agregar, eliminar y marcar tareas como completadas, además de un diseño responsivo que se adapta a diferentes tamaños de pantalla.

## Características

- **Agregar Tareas**: Permite a los usuarios introducir nuevas tareas con una categoría y prioridad.
- **Marcar como Completadas**: Incluye la opción de marcar tareas como completadas con un simple clic en un checkbox.
- **Eliminar Tareas**: Los usuarios pueden eliminar tareas que ya no necesiten.
- **Estilo Responsivo**: Diseñado para funcionar en dispositivos móviles y de escritorio.
- **Historial de Tareas**: Guarda el estado de cada tarea y permite visualizar el progreso.

## Tecnologías Utilizadas

- **HTML5**: Para la estructura del contenido.
- **CSS3**: Para el diseño y la presentación de la aplicación.
- **JavaScript**: Para manejar la lógica funcional y la interacción con el usuario.
- **LocalStorage**: Para el almacenamiento persistente de las tareas.

## Uso

1. **Agregar Tareas**: Escribe el nombre de la tarea y selecciona una fecha, categoría y prioridad, luego haz clic en "Agregar".
2. **Marcar Tareas Como Completadas**: Usa el cuadro de verificación junto a la tarea.
3. **Eliminar Tareas**: Haz clic en el ícono de papelera para eliminar una tarea.
